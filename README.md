# Simulated Intent Based Programming

This Python example simulates the core concept of Intent-Based Programming. It demonstrates how a high-level, natural language "intent" (what a developer wants to achieve) can be translated into specific, lower-level actions or simulated code generation (how to achieve it). The script uses simple regex matching to interpret user input and trigger predefined functions, illustrating the principle of abstracting away implementation details.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`
3. Enter intents like "create user with name Alice and email alice@example.com", "list active products", "calculate the sum of 10 and 5", or "send a welcome email to Bob".
